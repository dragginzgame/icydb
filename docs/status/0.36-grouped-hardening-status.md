# 0.36 Status (as of 2026-03-01)

## 0.36 Scope

This status tracks grouped execution hardening and aggregate-surface expansion
work for `0.36.x` from:

- `docs/design/0.36-ordered-group-execution.md`
- `docs/design/0.36-having.md`

`0.36` extends grouped execution foundations from `0.35` with ordered grouping
strategy formalization, HAVING support, DISTINCT aggregate semantics, streaming
eligibility enforcement, and grouped projection contract hardening.

---

## In Scope (`0.36.x`)

- ordered vs hash grouped execution strategy routing
- planner detection for ordered index-prefix grouped eligibility
- HAVING logical planning, validation, and grouped-stage execution
- DISTINCT aggregate support with deterministic semantics
- streaming eligibility matrix formalization and enforcement
- grouped projection alias/schema contract formalization
- grouped continuation shape/version tracking for HAVING-aware grouped queries
- release bookkeeping alignment in status/changelog docs

## Out of Scope (`0.36.x`)

- window functions
- nested multi-stage grouping pipelines
- storage-layer grouped pushdown
- parallel or distributed grouped execution
- user-defined aggregate plugins
