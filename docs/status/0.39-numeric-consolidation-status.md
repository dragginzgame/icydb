# 0.39 Status (as of 2026-03-02)

## 0.39 Scope

This status tracks numeric consolidation work for `0.39.x` from:

- `docs/design/0.39-numeric-consolidation.md`
- `docs/changelog/0.39.md`

`0.39` is a correctness-first line focused on converging numeric semantics
across planner typing, runtime expression behavior, aggregate eligibility, and
identity stability guardrails.

---

## In Scope (`0.39.x`)

- shared numeric capability/coercion contract extraction
- planner numeric typing closure (fail-closed where decidable)
- scalar/grouped runtime numeric evaluation convergence
- aggregate numeric semantics alignment with planner contracts
- continuation/fingerprint numeric drift guards
- structural guard hardening for numeric semantic ownership
- release bookkeeping in changelog/status docs

## Out of Scope (`0.39.x`)

- joins
- window functions
- cost-based planning
- expression pushdown into storage/index execution
- dynamic function/operator registration
- relational expansion beyond single-entity algebra
