# 0.45 Status (as of 2026-03-07, kickoff established)

## 0.45 Scope

This tracker covers cleanup work defined in:

- `docs/design/0.45-feature-cleanup.md`
- `docs/design/0.45-executor-planner-overlap.md`

Objective: close existing correctness/alignment/maintainability gaps without
adding new query-surface features.

---

## In Scope (`0.45.x`)

- API consistency cleanup (visibility, error surfaces, serialization stability)
- schema correctness cleanup (relation naming and compile-time validation)
- internal maintainability cleanup (overlap removal, file/module gravity-well splits, test rationalization)
- deterministic stability closure (fingerprint, cursor, explain output, ordering assertions)

## Out of Scope (`0.45.x`)

- new query terminals
- new planner semantic families
- new access-path variants
- new executor routing branches
- compatibility shim preservation
