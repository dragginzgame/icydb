# 0.38 Status (as of 2026-03-02)

## 0.38 Scope

This status tracks pre-unified-expression structural hardening work for
`0.38.x` from:

- `docs/design/archive/0.38-projection-expression-spine/0.38-design.md`
- `docs/changelog/0.38.md`

`0.38` is a consolidation line focused on projection authority, semantic
ownership boundaries, fingerprint isolation, and runtime error-boundary
hardening before larger modeling changes.

---

## In Scope (`0.38.x`)

- projection authority consolidation behind one semantic construction spine
- planner-vs-executor semantic validation deduplication
- aggregate fingerprint isolation from explain/alias metadata
- runtime class/origin preservation through query and facade conversion
- structural guard hardening for projection and semantic boundary drift
- release bookkeeping in changelog/status docs

## Out of Scope (`0.38.x`)

- unified `Expr` rollout as public query surface
- type-inference expansion
- continuation token format/version changes
- storage pushdown expansion
- projection surface expansion beyond current aggregate/projection contracts
