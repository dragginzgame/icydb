# 0.63 Status Tracker

Current state: near closure. `0.63` landed as a bounded consolidation line
across query/session boundaries, projection/materialization ownership,
mutation/query-plan authority cleanup, and a final generated/test surface
minimization pass.

## Scope

This status tracks the `0.63` mega-audit line defined in:

- `docs/design/0.63-mega-audit/0.63-design.md`
- `docs/design/0.63-mega-audit/0.63-addendum.md`

The goal remains consolidation, not feature expansion:

- collapse same-layer duplication
- keep one owner per authority boundary
- preserve the current wasm/runtime baseline while reducing drift
