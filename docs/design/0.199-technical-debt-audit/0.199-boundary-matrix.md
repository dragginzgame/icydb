## 0.199.0 Scope

0.199.0 starts the technical-debt audit line by anchoring the read/query
boundary matrix after the 0.197 and 0.198 hard cuts. It does not reopen
read-intent API design, exact-key canonicalization, public read admission,
cursor formats, persisted formats, or executor route behavior.

The first implementation decision after this artifact should be based on
evidence from the matrix below, not on a broad cleanup impulse.
