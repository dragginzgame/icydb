  silently falling back to zero randomness, while tests seed deterministic

  boundary types. `FieldRef`, `Response`, `CanonicalRow`, `VisitorIssues`,

  macros and narrowing runtime-only value operations, map helpers, semantics

- Replaced hidden raw-row and response coercions with explicit APIs such as
  `as_raw_row()`, `as_slice()`, `first()`, `last()`, `get()`, and
  `VisitorIssues::push()`.

### рџ§Є Testing
