# 0.101 Changelog

## 0.101.0

- `0.101.0` widens the projection-first scalar function surface without reopening `WHERE` again. The current line admits the new unary numeric functions on direct field arguments and the null/value-selection family on projection expressions, including nested `COALESCE(NULLIF(...), ...)` forms, while `COALESCE(...)` and `NULLIF(...)` in `WHERE` remain intentionally fail-closed for now.

```sql
SELECT name,
       COALESCE(NULLIF(guild_rank, background), class_name) AS preferred_label
FROM Character
ORDER BY preferred_label ASC, name ASC
LIMIT 20;
```
